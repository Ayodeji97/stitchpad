PTSP-11 — Order detail screen with multi-image

How should the detail screen present 2–3 style + 2 fabric images per item? Three options, all showing the same order with the same content state. Section 5 of the brainstorm proposed Variant A (hero = first image, small thumb strip below). B and C show what's possible if you want a richer photo-forward feel.

Variant A

Hero + thumb strip below

First image stays the big hero (160dp). Below it, a small strip shows the other style thumbs (52dp). Fabric strip lives in the garment card. Tap any thumb → fullscreen viewer.
