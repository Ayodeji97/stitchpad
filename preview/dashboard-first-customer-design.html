FirstCustomer Dashboard — proposed combined design

Combining the strongest patterns from the three ChatGPT mockups: shipped prominent hero (BrandNew-paired), setup checklist with progress, and a customer-ready strip for instant access. Skipping redundant tile rows.

What I picked, and why

  • From A  The "Focus today" sparkle pill on the hero — already shipped via prominent palette.
  • From B  Setup checklist with 4 stacked steps + 25% progress bar. The current step is saffron-tinted with a chevron — the only step that's tappable. Future steps are muted; finished steps are green.
  • From B  Customer-ready strip — small pill with avatar + name + Message button. Lets the operator reach the customer they just added without leaving the dashboard.
  • Shipped  Work pipeline empty card with saffron backdrop — kept, copy nudged.
  • Skipped  A's "Quick actions" 3-tile (redundant with the checklist). B's "What happens next" tiles (duplicates checklist labels). C's "Why this matters" tiles (educational but bloats the screen).

How this maps to existing code

  • Hero — `IllustratedFocusCard` with `prominent=true`, FirstOrder variant. Already shipped. Just a copy refinement (drop "is ready" wording, mention all three remaining steps).
  • Setup checklist — new component `SetupChecklistCard` in feature/dashboard/presentation/components/. Pure UI, takes a list of `ChecklistStep(label, description, status: Done/Active/Pending)`. The dashboard ViewModel computes the list from current state (customers exist? → step 1 done; orders exist? → step 2 done; etc.).
  • Customer-ready strip — new lightweight component, takes the single FirstCustomer's name + initials. "Message" CTA emits the existing WhatsApp-launch flow.
  • Work pipeline — already shipped. Empty card stays as-is, drop the "See how it works" link if we don't have target docs (keeps it visually quiet).
  • Hidden on FirstCustomer — weekly goal empty card, NBA empty card, reconnect strip. The setup checklist replaces the "what's next" guidance those previously fragmented across 3 cards.
