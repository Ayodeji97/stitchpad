Money & Today — four spins

The current dashboard renders nine vertically-stacked sections — a long scroll that buries the operator's two real jobs. All four variants below collapse the screen into a revenue hero + two-column split (do work / collect money) + pipeline strip. Each variant tests one design question. Pick one to take into Compose.

Shared thesis — the split

The dashboard answers two questions a tailor asks every morning: "what am I working on today?" and "who owes me money?" Stacking these into a long scroll forces the user to context-switch mid-thumb. Putting them side-by-side in two parallel columns lets the eye sweep both jobs in one glance — the dashboard becomes a desk, not a feed.

Top — HeroRevenue progress against weekly goal. Adapts to FocusVariant when the user hasn't earned a hero (e.g. brand-new, ready-for-pickup).
Middle — SplitTwo parallel columns. Left = today's work (overdue / due / ready). Right = money to collect (NBA collect prompts).
Bottom — PipelineCompact strip with a 5-block status spark. Tap → drill into the full Orders tab.
