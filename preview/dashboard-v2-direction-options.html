About the illustrations you see here

The mannequin/measuring-tape/red-fabric SVG in each hero is a placeholder I authored by hand — recognizable enough to evaluate the layout, but not the warm hand-drawn art from the GPT mockups. I cannot generate raster illustrations. Once you pick a direction, the realistic path is: I write a single style prompt, you generate ~6–8 illustrations in ChatGPT (one per state — busy/pickup/money/steady/quiet/welcome + 2 empty states), and we swap them in. Free, fast, consistent if we lock the prompt.

Tell me which option (A, B, or C), and I'll commit it to the design spec and start the implementation plan.
