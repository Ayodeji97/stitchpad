Comparison:
  • A — Speed-dial (recommended): 2 taps (FAB → Measurements), labeled text, always reachable at any scroll position, zero layout change to the live dashboard, and it reuses the exact SpeedDialAction pattern + the picker sheet that already shipped. Precedent: Inspiration already has this dual placement (dial + bottom Quick access). Implementation ≈ one action entry + one string.
  • B — Pinned chips: 1 tap and always visible, but permanently costs a row of vertical space on the revenue-first screen — every state pays, including users who came to check money. Also a brand-new UI pattern to maintain.
  • C — Header icon: 1 tap, free space-wise, but icon-only (rejected twice for ambiguity) and the header corner becomes 4 targets. Truncates the greeting sooner on long names.
  • All three open the SAME customer-picker sheet from PR 3 — the options differ only in the entry affordance.
  • The bottom Quick access section stays in all options (discoverability for scrollers), mirroring Inspiration's existing dual placement.
